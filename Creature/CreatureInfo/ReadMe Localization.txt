On Localization

Whenever you see a folder named "en" (or other lowercase letter shorthands, listed below) it's a localization folder. These contain text for different languages. So the "en" folder contains the english translation for text in your mod. 

These folders often contain 1 .xml or .txt file with the text that the game loads for that language. The folder name tells the game which language it is supposed to be.

Possible languages:

bg		Bulgarian
cn		Simplified Chinese
cn_tr		Traditional Chinese
en		English
es		Spanish
fr		French
jp		Japanese
kr		Korean		
pt_br		Brazilian Portuguese	
pt_pt		Portuguese		
ru		Russian
vn		Vietnamese


The files in these across different language folders have the same structure but different text content.
