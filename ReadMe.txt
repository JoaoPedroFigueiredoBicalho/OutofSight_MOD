This skeleton is intended to help new modders get an abno mod up and running, so that they can start adapting it for their abno concept.  As such, it does not go into detail about how or why things work. 
This is NOT intended to be used in actual gameplay.

XML Files:

Mods in LC have a lot of information in XML. (basically a way to structure information so your PC can read it) While this mod has comments on what the data in the xmls does, it won't help you understand how xml works itself. If you don't know xml, I recommend https://www.w3schools.com/xml/xml_whatis.asp
Basically, only edit values that are either between tags like <tag> value </tag> and not the tags itself, or values after an equals something = "value" 
The portions starting with <!-- and ending with --> are my comments, you can leave those alone.
If edited a xml file, I recommend running it through an xml validator online, to make sure you didn't accidentally delete a < or >

I will be using the word Abno as shorthand for Abnormality.

There are more functionalities in basemod, such as the ability to load custom sounds, assets, or animations for EGO weapons and employees, but these are excluded here as this is supposed to be the very barebones emptied out version of an abnormality.
The creation of code is also no covered here, because it is too varied to be feasibly covered in a mod like this. More info for that is in the guide.
