# Out of Sight, Out of Mind - Lobotomy Corporation Mod

A custom abnormality mod for Lobotomy Corporation implementing F-02-152.

## Overview

This mod adds an abnormality with a three-employee marking system:
- **First employee**: Gets "Mindful" effect (BLACK damage resistance)
- **Second employee**: Gets "Sightful" effect (PALE damage resistance) and both begin attacking each other
- **Third employee**: All three marked employees die

## Installation

Extract to `LobotomyCorp_Data/BaseMods/`

## Building

**Windows:**
```batch
build.bat
```

**Linux:**
```bash
./build.sh
```
